## Summary
- `cmoc apply fork` の差分要約・所見列挙・所見受け取り修正の 3 つの出力契約と、それぞれの agent call 組み立てを扱う入口。fork 適用後の変更を要約したい、問題点を列挙したい、所見を受けて修正依頼したいときにここから下位ファイルへ進む。
- 個別の判定ロジックや共通プロンプト生成ではなく、fork 配下の役割分担と各ファイルを読む境界を確認するためのルーティング対象。

## Read this when
- `cmoc apply fork` の変更差分を要約・レビュー・修正依頼のどれかに振り分ける入口を確認したいとき。
- 差分要約、ファイル単位の所見列挙、所見を受けた実装修正のいずれかの出力契約や呼び出し条件を確認したいとき。
- fork 配下の各ファイルの役割分担を知り、次に読むべき下位ファイルを選びたいとき。

## Do not read this when

- 差分取得、fork 作成、適用フロー全体の実行制御を調べたいときは、この配下ではなく上位の apply 側を読む。
- 共通の prompt 生成規約や別サブコマンドの仕様を探しているとき。
