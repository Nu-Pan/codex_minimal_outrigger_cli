# `oracle`

## Summary
- `cmoc review oracle` の所見処理に関する prompt 生成と Structured Output schema をまとめて扱う入口。新規所見の列挙、所見の整理、妥当性理由の収集、採否判定のいずれかを変えたいときに読む。

## Read this when
- 新規所見をどう列挙し、既知の関連所見とどう重複排除するかを確認したいとき。
- 所見リストの重複解消や統合の方針を確認したいとき。
- 所見が妥当である理由・妥当ではない理由を、既知理由と重複しない形で集める契約を確認したいとき。
- 所見を人間へ提示すべきかどうかの判定結果と、その出力契約を確認したいとき。

## Do not read this when
- `cmoc review oracle` 全体の実行順や CLI の入出力だけを追いたいときは、呼び出し元の実行処理を読む。
- oracle file 全般の品質基準や、何を問題所見とみなすかという原則だけを確認したいときは、各種 oracle standard 側を読む。
- 個別の所見本文やレビュー対象の具体的な内容だけを確認したいときは、この層ではなく対象 oracle file を読む。
