# `oracle`

## Summary
- `cmoc review oracle` の所見を oracle file に基づいて評価・整理するための review/oracle 配下の入口です。ここでは、所見が妥当かどうかの根拠を新規に挙げる側、妥当性を否定する側、複数所見を重複・矛盾の観点で整理する側、そしてそれらの結果を読みやすい形にまとめる側を扱います。
- 個別のファイルは、所見の列挙、擁護、反証、統合のどの段階を見たいかで分かれています。新しい所見を探すなら列挙側、既知の所見に対する支持理由や反証理由を確認するならそれぞれの判定側、既出所見をまとめ直すなら統合側を読むのが適切です。
- oracle file の内容そのもの、review 全体の実行フロー、あるいは別サブコマンドの prompt 組み立てはこの階層の主対象ではありません。所見の探索手順や上位コマンドの呼び出し仕様を追うなら、より上位の実装か各担当ファイルへ進むべきです.

## Read this when
- oracle file レビューで、新規所見の列挙・採否判定・擁護理由・反証理由・統合判断のどれかを変えたいとき。
- 所見一覧の出力契約や、所見ごとの根拠の持たせ方を確認したいとき。
- レビュー済み所見を重複や矛盾の観点で整理する処理を追いたいとき。
- この review/oracle 配下で、どの責務がどのファイルに分かれているかを確認したいとき.

## Do not read this when
- oracle file の個別内容や記述方針そのものを知りたいだけのとき。
- `cmoc review oracle` の上位コマンド実行フローや、レビュー対象の探索手順を追いたいとき。
- 別の review 配下の用途ではなく、一般的な prompt builder や別サブコマンドのパラメータ生成を探しているとき。
- INDEX.md エントリーの生成規則やルーティング文書そのものを確認したいとき。
