- `acp_builder` は参照可能な正本ソースがない空の入口です。
- `other` は設定・パスモデル・規範データ構造・構造化文書レンダリングを扱います。

## Read this when
- 正本ソースの有無を確認するときは `acp_builder` を読む。
- 設定値、パス解決、規範データ構造、Markdown レンダリングを調査するときは `other` を読む。
