# `oracle`

## Summary
- `oracle/src/oracle` 配下の仕様断片と生成入口を束ねるルーティング対象。共通定義と各サブ領域の責務分担を見分けたいときの入口で、個別の正本本文や実装詳細をここで完結させない。
- 共通の呼び出しパラメータ型、設定、パス解決、標準文レンダリングの基盤を確認したいときは、この階層から対応する下位定義へ進む。
- `cmoc apply`、`cmoc indexing`、`cmoc review`、`cmoc session join`、`cmoc tui` 向けの agent 呼び出し正本を探すときは、この階層からサブコマンド別領域へ進む。

## Read this when
- oracle 側の共通型や基盤定義を探していて、まず読む対象を切り分けたいとき。
- 特定サブコマンドの prompt 生成や Structured Output の入口を探していて、共通定義と個別定義のどちらを読むべきか判断したいとき。
- 設定、パス解決、標準文レンダリングのいずれかを変更したくて、その責務の所在を確認したいとき。

## Do not read this when
- 個別サブコマンドの prompt 本文、出力契約、実行条件を確認したいときは、対応する下位領域を直接読む。
- 共通の設定定義、パス解決、標準文レンダリングの詳細だけを確認したいときは、対応する定義元を直接読む。
- agent 呼び出しの実行フロー全体や CLI 側の制御を追いたいときは、この階層ではなく実行側を読む。
