# `oracle`

## Summary
- oracle/src/oracle 配下の正本ソースをまとめるディレクトリ。ACP 呼び出し設定、cmoc の設定・パスモデル・規範データ構造、プロンプト構築と標準ルール部品の各領域への入口を提供する。

## Read this when
- ACP 呼び出し設定や呼び出し条件に関係する正本ソースを探索するとき。
- cmoc の設定値、パス解決、規範文書の構造化・Markdown 変換を調査するとき。
- プレースホルダ展開、完全なエージェントプロンプトの構築、プロンプトへ注入する標準ルールを調査・変更するとき。

## Do not read this when
- 具体的な CLI 機能や入出力処理だけを調査するとき。
- 個別の prompt builder 部品、個別標準ルール、個別ファイル操作の詳細だけを確認したいときは、該当する配下ファイルを直接読む。
