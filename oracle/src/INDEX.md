# `oracle`

## Summary
- `oracle/src/oracle/acp_builder` 配下の、ACP builder 全体で複数の実装にまたがって共有される状態・ルール・結果表現を読むための入口。個別 builder の手順ではなく、共通概念の境界を先に確かめたいときにここから進む。
- `oracle/src/oracle/other` 配下の、cmoc の設定モデル、ルートパス解決、規範文書のデータ化、構造化 markdown 組み立てを扱う入口。設定や文書表現などの共通基盤を確認したいときに使い、CLI 手順や個別業務ロジックは直接の対象にしない。
- `oracle/src/oracle/prompt_builder` 配下の、プレースホルダ型、完全なプロンプト生成、oracle/realization の規範文面生成を分けて扱う入口。プロンプト本文や規範文面を構成する共通材料の責務境界を確認したいときに読む。

## Read this when
- ACP builder 全体で共通して使う状態、ルール、結果表現の正本仕様断片を確認したいとき。
- ACP builder の複数領域にまたがる挙動を実装・テストへ反映する前に、共有概念の境界を確認したいとき。
- cmoc の永続設定モデルや、その既定値・保存方針を確認したいとき。
- ルートパスの解決や、実パスとの相互変換を確認したいとき。
- 規範文書を保持するデータ構造や、markdown への落とし込み方を確認したいとき。
- 階層付き markdown のレンダリングや、`cmoc_ref` と `cmoc_block` の検査、文字列やコードブロックの正規化を確認したいとき。
- プレースホルダ展開で使う型の意味を確認したいとき。
- 完全な agent 用プロンプトの構成や、静的な規約と可変要素のまとめ方を確認したいとき。
- oracle と realization の基本区分や、各種標準文面の責務境界を確認したいとき。

## Do not read this when
- 特定の builder の個別仕様だけを確認したいとき。
- realization code 側の実装詳細やテスト構成を確認したいとき。
- ACP builder と無関係な oracle src の仕様を探しているとき。
- `cmoc` の各サブコマンドの実行手順や CLI の入出力を追いたいとき。
- 設定読み書きの具体的な永続化処理や、サブコマンド固有の業務ロジックを見たいとき。
- 構造化文書やパス解決以外の別系統の共通処理を探したいとき。
- プロンプト本文の生成手順や置換ロジックの詳細だけを知りたいとき。
- 個別の標準本文そのものを確認したいとき。
- 実装コードやテストコードの修正方針だけを確認したいとき。
