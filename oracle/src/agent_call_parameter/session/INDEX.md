# `join`

## Summary

- `cmoc session` 系サブコマンドの Codex CLI 呼び出し仕様への入口です。
- `join/` は `cmoc session join` の merge conflict marker 解消用 agent call parameter をまとめます。

## Read this when

- `cmoc session join` で発生する Codex CLI 呼び出し仕様を確認したいとき。
- session 系サブコマンドのうち、どの下位ディレクトリへ進むべきか判断したいとき。

## Do not read this when

- `cmoc session join` 以外の session 状態管理や git 操作だけを確認したいとき。
- apply、review、indexing の agent call parameter を探しているとき。
