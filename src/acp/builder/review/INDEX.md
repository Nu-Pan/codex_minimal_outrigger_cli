# `oracle`

## Summary
- oracle レビュー向け ACP 呼び出しビルダーと、その Structured Output schema 群をまとめる領域。新規所見の列挙、所見の擁護・反証、採否判定、重複や矛盾を含む所見リストの整理について、AI エージェントへ渡す prompt・補助文脈・ファイルアクセス制約・モデル設定と、各段階の出力契約への入口になる。
- レビュー対象は oracle file とその正本仕様断片であり、生成・検証・判定・整理の各ステップで、既知理由や既存所見との重複を避けながら、人間へ提示すべきレビュー所見を絞り込むための部品を扱う。

## Read this when
- `cmoc review oracle` のレビュー所見フローで、新規所見の列挙、所見の妥当性を支える理由の追加調査、所見を否定する理由の追加調査、採否判定、所見リストの整理のいずれかに関する ACP 呼び出しパラメータを確認または変更したいとき。
- oracle file を根拠にしたレビュー処理で、対象所見・既知の擁護理由・既知の反論理由・既存所見群がどのように prompt の補助文脈へ渡されるかを追いたいとき。
- レビュー所見関連の Structured Output schema を探しており、所見列挙、擁護理由、反証理由、採否判定、マージ編集操作のどれを読むべきか切り分けたいとき。
- レビュー用エージェントに対するファイルアクセス制約、利用モデル、推論強度、oracle 標準や review oracle 標準の組み込み箇所、隣接 schema への接続を確認したいとき。

## Do not read this when
- `cmoc review oracle` の CLI 引数解析、サブコマンド定義、実行順序、保存、表示、集計など、レビュー所見フロー全体の上位制御を調べたいとき。
- oracle file や realization file の基本定義、oracle 標準、review oracle のレビュー基準そのものを確認したいとき。
- 具体的な oracle file の仕様内容や、所見に基づく oracle file の編集提案を調べたいとき。
- レビュー以外の ACP 呼び出しビルダー、共通 prompt 部品、汎用 JSON Schema の書き方、INDEX.md エントリー生成規則を確認したいとき。
