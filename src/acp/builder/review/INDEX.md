# `oracle`

## Summary
- oracle レビューにおける AI 呼び出しパラメータ構築と Structured Output 契約をまとめた領域。正本仕様断片を起点にした新規所見列挙、所見の擁護・反証理由列挙、採否判定、重複・矛盾の整理を扱う。
- 各レビュー段階について、prompt に渡す補助文脈、oracle 読み取り制約、モデル・推論設定、隣接する出力契約への接続を確認する入口になる。

## Read this when
- `cmoc review oracle` の所見生成、検証、採否判定、マージに使う AI 呼び出しの役割・目的・入力文脈・制約を確認または変更したいとき。
- レビュー所見を機械処理できる形で返すための出力契約を、処理段階ごとに確認・実装・テストしたいとき。
- 既知の所見や理由と重複しない新規所見・新規擁護理由・新規否定理由を返させる制御を追いたいとき。
- oracle file の記述だけを根拠にしたレビュー結果を、列挙、検証、採否、整理のどの段階でどう扱うかを調べたいとき。

## Do not read this when
- oracle file や realization file の基本概念、正本仕様断片としての一般ルール、レビュー基準そのものを確認したいとき。
- `cmoc review oracle` の CLI 引数解析、サブコマンド実行順序、所見の保存・表示・集計など、AI 呼び出しパラメータ構築より外側の実行制御を調べたいとき。
- 個々の oracle file の仕様内容や、所見に基づく oracle file の具体的な編集案を確認したいとき。
- レビュー以外の ACP 呼び出しパラメータ構築、共通プロンプト部品、汎用的な JSON Schema 構文を調べたいとき。
