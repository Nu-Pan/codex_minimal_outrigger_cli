# `oracle`

## Summary
- 正本仕様断片レビューの各段階で使う AI 呼び出しパラメータ構築と、その Structured Output schema をまとめた領域。新規所見の列挙、所見の擁護・反証理由の追加調査、採否判定、重複・矛盾する所見の整理について、プロンプト入力、ファイルアクセス制約、モデル・推論設定、出力契約への接続を扱う。
- レビュー処理全体ではなく、oracle file を根拠にした所見生成・検証・判定・整理をエージェントへ依頼する境界と、各依頼の機械的な返却構造を確認するための入口である。

## Read this when
- 正本仕様断片レビューで、新規所見を列挙するエージェント呼び出しや、その所見一覧の出力契約を確認・変更したいとき。
- レビュー所見が妥当である理由、または妥当ではない理由を、既知理由と重複しない形で oracle file から追加調査させるプロンプトや出力契約を確認したいとき。
- レビュー所見を人間に提示するか却下するかの採否判定について、判定用プロンプト、補助入力、出力 schema との対応を追いたいとき。
- 複数のレビュー所見に含まれる重複や矛盾を、削除・置換・統合などの整理操作へ変換させる呼び出し内容や出力形式を確認したいとき。
- 所見本文、既知の擁護理由、既知の反論理由、レビュー対象の正本仕様断片パスなどが、レビュー用 AI 呼び出しの補助文脈へどう組み込まれるかを調べたいとき。
- レビュー用 AI 呼び出しで使うファイルアクセスモード、モデル種別、推論強度、標準プロンプト断片、Structured Output schema 接続を段階別に確認したいとき。

## Do not read this when
- 正本仕様断片そのものの内容、oracle file の一般定義、レビュー基準、oracle 標準を確認したいだけのとき。
- レビュー全体の CLI 引数解析、サブコマンド実行順序、所見の保存・表示・集計など、呼び出しパラメータ構築より上位の実行フローを調べたいとき。
- INDEX.md エントリー生成やルーティング文書の書き方を調べたいとき。
- レビュー後に人間へ提示された所見の UI、通知、永続化形式などを確認したいとき。
- 汎用的な JSON Schema の構文、または oracle レビュー以外の ACP 呼び出しパラメータ構築を調べたいとき。
- 個々の oracle file に対する具体的な仕様修正案、実装方針、テスト方針を確認したいとき。
