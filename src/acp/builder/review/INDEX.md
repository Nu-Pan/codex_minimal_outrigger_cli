# `oracle`

## Summary
- `cmoc review oracle` 向けに、レビュー所見の列挙、妥当性検証、採否判定、重複・矛盾整理を行う AI 呼び出しパラメータ構築と、それぞれの Structured Output schema をまとめる領域。
- oracle file を根拠にしたレビュー所見ワークフローのうち、個別エージェントへ渡す prompt、補助文脈、ファイルアクセス制約、モデル設定、schema 接続を確認する入口になる。
- 同階層には、所見生成・擁護理由列挙・否定理由列挙・採否判定・所見マージの各処理について、呼び出し構築実装と出力契約が対で配置されている。

## Read this when
- `cmoc review oracle` の所見列挙、所見検証、採否判定、所見整理に関する AI 呼び出し内容や出力契約を確認・変更したいとき。
- レビュー所見に対して、oracle file を根拠にした新規所見、妥当理由、否定理由、採否判断、重複・矛盾解消操作をどのように構造化して返すかを調べたいとき。
- 既知の所見や既知理由との重複を避けるための補助入力、純粋 oracle 読み取りの制約、レビュー用標準プロンプトの組み込み方を追いたいとき。
- レビュー処理の各段階で、実装側の呼び出しパラメータと隣接する JSON schema の対応関係を確認したいとき。

## Do not read this when
- oracle file や realization file の基本概念、oracle 標準、review oracle 標準そのものを確認したいだけのとき。
- `cmoc review oracle` の CLI 引数解析、サブコマンドの実行順序、所見の保存・表示・集計など、AI 呼び出しパラメータ構築より上位または後段の処理を調べたいとき。
- 個々の正本仕様断片の内容や、所見に従った oracle file の具体的な編集案を確認したいとき。
- INDEX.md エントリー生成、ルーティング文書の書き方、汎用的な JSON Schema 構文を調べたいとき。
