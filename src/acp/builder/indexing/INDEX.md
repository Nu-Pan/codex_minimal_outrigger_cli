# `index_entry.py`

## Summary
- `cmoc indexing` の目次情報生成で、対象ファイルまたはディレクトリの内容から `INDEX.md` 用エントリーを生成させるための AI 呼び出しパラメータを組み立てる実装。
- 対象パスを実パスへ解決し、読み取り専用の完全プロンプトに役割・目的・エントリー生成規則・対象本文を埋め込み、効率重視モデルと低い推論負荷、出力 schema パスを指定した呼び出し設定として返す。

## Read this when
- `cmoc indexing` が個別対象の `INDEX.md` エントリー生成プロンプトをどのように構築しているか確認したいとき。
- エントリー生成時に既存 `INDEX.md` を読ませず、対象本文と必要な関連文書だけを根拠にさせる制約を調整したいとき。
- 目次情報生成用の `AgentCallParameter` に渡すモデル種別、推論負荷、ファイルアクセスモード、Structured Output schema の指定を確認したいとき。

## Do not read this when
- `INDEX.md` 全体の走査、更新、保存、ファイルシステム上の生成処理を調べたいとき。
- 生成されたエントリーの schema 定義や markdown レンダリングの詳細を調べたいとき。
- 通常の実装変更やテスト追加で、目次情報生成プロンプトの呼び出し設定に関係しないとき。
