# `builder`

## Summary
- AI エージェントへ渡す呼び出しパラメータを、用途別に構築する実装群をまとめる領域。apply 後レビュー、ルーティング文書エントリー生成、oracle レビュー、Codex セッション依頼文、TUI 実行前パラメータ解決など、サブコマンドや実行段階ごとの prompt、補助文脈、ファイルアクセス制約、モデル設定、Structured Output 契約を扱う。
- 個別のエージェント呼び出し内容を調べるときは、対象サブタスクに対応する下位領域へ進む入口になる。共通の呼び出し実行そのものではなく、各用途でどの役割・目的・標準文書・入力情報・出力 schema をエージェントに渡すかを確認するための領域。

## Read this when
- cmoc の各機能が AI エージェントに渡す依頼文、補助指示、読み書き権限、モデル種別、推論強度、Structured Output schema を用途別に確認または変更したいとき。
- apply 後の差分要約、realization file 所見列挙、検出済み所見の修正依頼に関するエージェント呼び出しパラメータを追いたいとき。
- INDEX.md エントリー生成で、対象本文の扱い、関連文書参照、読み取り専用条件、構造化出力契約がどう組み立てられるか確認したいとき。
- oracle レビューで、所見列挙、理由調査、採否判定、所見整理などの段階ごとに AI へ渡す文脈と出力契約を確認したいとき。
- Codex セッション起動用プロンプトに、対象ファイル、標準仕様、追加指示、ファイルアクセス規則、role や goal がどう統合されるか確認したいとき。
- TUI 実行前に、元プロンプトからファイルアクセスモードや標準参照要否を判定するエージェント呼び出しを確認したいとき。

## Do not read this when
- エージェント呼び出しの共通実行基盤、プロンプト部品の汎用レンダリング、JSON Schema 読み込み、path model、構造化ドキュメント表現そのものを調べたいとき。
- CLI 引数解析、サブコマンドの実行順序、git 操作、状態保存、結果の表示や集約など、各機能の外側の制御フローを調べたいとき。
- oracle file、realization file、各種 standard 文書など、prompt に差し込まれる正本仕様や標準本文そのものを読みたいとき。
- レビュー所見や変更要約の実際の判断基準、カテゴリ分類、git diff 生成、対象ファイル抽出アルゴリズムを調べたいとき。
- TUI の画面表示、対話フロー、エディタ入力処理、コメント除去など、UI や入力処理の詳細を確認したいとき。
- 生成済み INDEX.md の内容やルーティング文書としての良し悪しだけを確認したいとき。

# `prompt_parts`

## Summary
- AI agent に渡すプロンプトを構成する部品群を扱うディレクトリ。ファイルアクセス規則、ルーティング規則、oracle/realization の基本説明、oracle・realization・レビュー・INDEX.md エントリーの標準文書を構造化文書として組み立てる実装がまとまっている。
- 個別の標準文書本文を生成する部品と、それらを依頼内容・アクセスモード・追加プロンプト・有効化フラグに応じて完全な agent 用プロンプトへ集約する入口がある。

## Read this when
- AI agent に渡す最終プロンプトの構成、標準文書の注入順序、または有効化フラグ間の依存関係を確認・変更したいとき。
- ファイル読み書き制約、INDEX.md を使った読み進め方、oracle file と realization file の責務境界を、agent 向けプロンプトとしてどの文面で提示しているか確認したいとき。
- oracle file、realization file、oracle review、oracle-to-realization review、INDEX.md エントリー生成に関する標準文書の生成内容を確認・変更したいとき。
- 標準文書を構成する要求・背景・判断例が、どの構造化文書として agent prompt に組み込まれるかを追いたいとき。

## Do not read this when
- 特定の CLI サブコマンド、状態ファイル、path model、入出力 schema などの個別機能仕様や実装詳細を探しているとき。
- 標準文書を利用する側ではなく、構造化文書型、標準項目型、またはそれらの変換基盤そのものを確認したいとき。
- テスト実装や fixture の内容を確認したいとき。
- INDEX.md エントリーの完成文面だけが必要で、対象本文をすでに直接読める状態にあるとき。
