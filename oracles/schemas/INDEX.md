# `structured_output`

## Summary

- `<cmoc-root>/oracles/schemas/structured_output/` 配下の Structured Output schema をまとめた目次です。
- 現状は `review/` を案内し、`cmoc review oracles` で使う JSON schema 群へ分岐します。
- 所見の列挙・統合・妥当性検証・採否判定の各段階に応じて、読むべき schema を切り分ける入口です。

## Read this when

- Structured Output schema の置き場所と役割分担を確認したいとき。
- `review/` 配下の schema 一式へ進むべきか迷ったとき。
- `cmoc review oracles` 用の JSON schema を追加・修正・レビューしているとき。

## Do not read this when

- `review/` 配下の個別 schema がすでに分かっていて、直接読むとき。
- `cmoc review oracles` の利用手順や評価フローだけを確認したいとき。
- Structured Output ではなく、`docs/` 配下の自然言語仕様や開発規約を探しているとき。
